# Workplace Training Module

An activity module designed to streamline workplace training evaluations and assessments. This plugin enables instructors to create structured training programs with customizable
sections and items, allowing for detailed evaluation and tracking of workplace training.

## Use Cases

- Structured competency assessments
- Practical evaluations
- Work-integrated learning assessments

## Basic Usage

### Creating a Workplace Training Activity

1. Turn editing on in your course
2. Click "Add an activity or resource"
3. Select "Workplace Training"
4. Configure the activity settings and save

### Managing Training Sections

1. Open the Workplace Training activity
2. Click "Manage Sections" to add, edit, or delete sections
3. Add items to each section

### Evaluating Students

1. Access the Workplace Training activity
2. Click "Evaluate Students"
3. Select a student to evaluate
4. Complete the evaluation items for each section
